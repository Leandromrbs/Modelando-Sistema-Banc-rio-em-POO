# Desafio Sistema Bancário

## Modelando o Sistema Bancário em POO em Python

### O sistema deverá realizar as seguintes funcionalides:<br><br>

 > - Depositar
 > - Sacar
 > - Extrato
 > - Nova conta
 > - Listar contas
 > - Novo usuário
 > - Sair

### Nota: Serão permitidos apenas três saques, limitado a R$500,00 cada.<br><br>
